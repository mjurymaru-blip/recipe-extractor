# YouTubeレシピ抽出PWA - プロジェクト概要

**作成日**: 2026-02-01  
**ステータス**: 企画・設計段階（レビュー反映済み）

---

## 🎯 このプロジェクトについて

YouTubeのレシピ動画から調理手順を**自動抽出**し、キッチンで使いやすい**ステップカード形式**で表示するPWA。

### 解決したい課題
- 動画はどんどん進んでしまい、調理中に追いつけない
- 一時停止・巻き戻しが面倒
- テキストだけのレシピは視覚的にわかりにくい

### ソリューション
- YouTube URLを入力、または字幕/概要欄をテキスト貼り付けでAIがレシピを構造化
- 自分のペースで進められるカード形式
- タップで動画の該当箇所にジャンプ可能
- 抽出結果は手動で編集・修正も可能

---

## 📱 想定する使い方

1. **レシピ登録時**: PCまたはスマホでYouTube URLを入力（または字幕テキストを貼り付け）、抽出・保存
2. **調理時**: キッチンでスマホ/タブレットを見ながら調理
   - 画面分割で動画を流しながら見ることも可能
   - ステップをタップして動画の該当箇所を確認

---

## 🛠️ 技術構成

| 項目 | 選択 |
|------|------|
| UI | Vanilla JS + CSS（PWA） |
| 字幕取得 | YouTube Transcript API（+テキスト貼り付けフォールバック） |
| レシピ解析 | Gemini API |
| ローカル保存 | IndexedDB |
| 永続保存 | JSON → GitHub |
| ホスティング | GitHub Pages |

---

## ⚠️ 想定リスク

| リスク | 対策 |
|--------|------|
| 字幕取得失敗 | テキスト貼り付けによるフォールバック |
| AI解析ミス | 編集機能をMVP時点で提供 |
| 自動字幕の精度不足 | 手動修正 + 問題発生時に都度対応 |
